Align all stakeholders and update your systems with the latest status of your purchase orders

  • Supply chain managers face a number of challenges in keeping up with vendor updates

  • Vendors share updates on the status of orders and offers via emails, text or phone. The supply-chain manager then needs to retrieve the relevant purchase orders, check what changed, and update Odoo.

  • Furthermore, she needs to manually check if the changes have any impact on existing sales orders and manufacturing orders. And if there is an impact, the relevant sales or production manager needs to be notified.

  • Doing this for every vendor communication can be a daunting task.

  • We built an Odoo app that seeks to automate these tasks by using the large language models behind ChatGPT. It reads and understands vendor emails, extracts relevant information about the order, assists users on how to update Odoo and notifies the relevant stakeholders.
